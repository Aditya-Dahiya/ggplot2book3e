Chapter 10

Position Scales and axes

Author

Aditya Dahiya

Published

December 23, 2023

This Chapter has no exercises to solve, so we use #TidyTuesday webpage’s data-set on R Package Structure (from the paper Historical Trends in R Package Structure and Interdependency on CRAN by Mark Padgham and Noam Ross) to explore and customize Position Scales and axes.

Libraries and Data

Code
library(tidyverse)     # Data wrangling and plotting
library(ggthemes)      # ggplot2 themes
library(patchwork)     # Composing multiple plots
library(ggiraph)       # Interactive ggplot2 graphs
library(ggstream)      # Stream graphs in R
library(paletteer)     # Huge color palettes aggregator
library(scales)        # Scales for plots
library(gganimate)     # Animations


# Loading Data
url <- 'https://raw.githubusercontent.com/rfordatascience/tidytuesday/master/data/2023/2023-12-26/cran_20221122.csv'
rpkgstats <- readr::read_csv(url)

# Focussing on number of lines of code (LOC) and files in R directory of packages
df1 <- rpkgstats |> 
  select(package, version, date, files_R, loc_R)

We can also demonstrate Visual range expansion in the following example exploring number of R packages updated or released over time on CRAN in Figure 2. We also demonstrate the use of scale_y_continuous(expand = expansion(0)) and scale_x_date(expand = expansion(0)) to remove any extra space between the graph and x-axis labels and y-axis labels.

Code

The best use of expand = expansion(0) comes with heat-maps. Here is an example in Figure 3 demonstrating the number of packages of R updated or released each month of the year, since 1999. We’ve made it interactive using ggiraph to make it more appealing to the user.

Code
library(ggiraph)

g1 <- df1 |>

Code
library(ggiraph)

g4 <- rpkgstats |> 
  mutate(license = as_factor(license)) |>

  mutate(year = year(date),
         month = month(date)) |> 
  group_by(year, license) |> 
  count() |> 
  group_by(year) |> 
  mutate(prop = n / sum(n),

anim <- df3 |>

10.3 Discrete Position Scales

Using limits, breaks and labels on discrete scales

Let’s use a data-set I found on Data is Plural (2023-12-20 edition) by Jeremy Singer-Vine on the Dengue Trends in India1 from the OpenDengue project (Clarke et al. 2023). The data definitions are available here.

The Figure 11 demonstrates the use of scale_y_discrete() to customize the position of, and the label for a specific value (“Other States”) on the y-axis.

Code
library(tidyverse)
library(here)
library(patchwork)

dengue <- read_csv(here("data", "dengue_india.csv"))

dendf <- dengue |> 
  rename(country = adm_0_name,
         state = adm_1_name,
         year = Year,
         cases = dengue_total,
         definition = case_definition_standardised) |> 
  select(country, state, year, cases, definition) |> 
  mutate(state = if_else(state == "ARUNACHAL\nPRADESH",
                         "ARUNACHAL PRADESH",
                         state))


df1 <- dendf |>
  filter(year == 2022) |> 
  group_by(state) |> 
  summarise(total = sum(cases, na.rm = TRUE)) |>
  ungroup() |> 
  mutate(
    state = str_to_title(state),
    state = factor(state),
    state = fct_lump_n(state, n = 10, w = total)
    ) 
  
label_state <- df1 |> 
  group_by(state) |> 
  summarise(total = sum(total)) |> 
  arrange(desc(total)) |> 
  pull(state)

g1 <- df1 |>

10.4 Binned position scales

The Figure 13 demonstrates a use for scale_x_binned() when we might want to use histogram like scale labels even for scatter-plots or bar plots. Here we show the number cases year-wise (between Jan 1 - Dec 31) of a year for each state, from 1991 to 2022. Since our x-axis variable is not a continuous one, the use of scale_x_binned() does not serve any purpose here, and this is not a good example.

Code
library(ggiraph)

g3 <- dendf |> 
  drop_na() |> 
  mutate(state = str_to_title(state)) |> 
  group_by(year, state) |> 
  summarize(cases = sum(cases)) |> 
  ungroup() |> 
  mutate(id = row_number()) |> 
  ggplot(aes(x = year, 
             y = state,
             size = cases,
             data_id = id,
             tooltip = paste0(state,
                              "\nYear: ", year,
                              "\nNumber of cases: ", cases),
             col = state)) +
  geom_point_interactive(alpha = 0.5) +
  labs(
    x = NULL, y = NULL,
    title = "Number of dengue cases is various states",
    subtitle = "Size of Dot represents number of cases. Hover on dots for details."
  ) +
  scale_x_binned(expand = expansion(),
                 guide = guide_axis(angle = 90),
                 n.breaks = 30) +
  theme_minimal() +
  theme(
    axis.ticks.x = element_blank(),
    axis.text.x = element_text(hjust = 0),
    legend.position = "none", 
    plot.title.position = "plot",
    plot.title = element_text(hjust = 0.5), 
    plot.subtitle = element_text(hjust = 0.5)
  )

So, let’s use scale_y_binned() in a plot where it’s use is better expressed, as in Figure 15. Here, we plots the number of R packages released each month vs. Lines of Code over the years in the form of a scatter plot in Figure 14, leading to a crowded plot with with limited ease of understanding. Next, we bin the y-axis, i.e. Lines of Code along with a log10 transformation, and break-up x-axis (dates) into quarters of the years. Now, the scatter plot in Figure 15 seems much easier to understand.

Code
df1 |> 
  drop_na() |> 
  filter(date > as_datetime("2015-01-01")) |> 
  ggplot(aes(x = date,
             y = loc_R)) +
  geom_point(alpha = 0.2) +
  labs(x = "Date of Release",
       y = "Lines of Code in R package",
       title = "Lines of Code in R packages updated / released over the years") +
  theme_minimal() +
  theme()

df1 |> 
  drop_na() |> 
  filter(date > as_datetime("2015-01-01")) |> 
  mutate(qtr = floor_date(date, unit = "quarter"),
         qtr = as_date(qtr)) |> 
  ggplot(aes(x = qtr,
             y = loc_R)) +
  geom_count(alpha = 0.2) +
  scale_x_date(breaks = scales::breaks_width("1 year", 
                                             offset = "-45 days"),
               date_minor_breaks = "3 months",
               date_labels = "%Y",
               expand = expansion(c(0.01, 0.05))) +
  scale_y_binned(trans = "log10",
                 labels = scales::label_number()) +
  scale_size(range = c(1, 10)) +
  labs(x = NULL,
       y = "Lines of Code in R package",
       title = "Lines of Code in R packages updated / released over the years",
       size = "Number of R packages") +
  theme_minimal() +
  theme(
    legend.position = "bottom",
    panel.grid.major.y = element_line(linetype = "dotted"),
    panel.grid.major.x = element_line(linetype = "dotted"),
    panel.grid.minor.x = element_blank(),
    plot.title = element_text(hjust = 0.5,
                              face = "bold"),
    axis.text.x = element_text(vjust = 1,
                               hjust = +2,
                               face = "bold"),
    plot.title.position = "plot"
  )
